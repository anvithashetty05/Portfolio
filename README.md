# 🌿 Anvitha J Shetty — Personal Portfolio

A clean, responsive personal portfolio website built with vanilla HTML, CSS, and JavaScript. No frameworks, no dependencies — just pure web fundamentals.

---

## ✨ Features

- **Multi-page navigation** — About, Skills, Projects, and Contact sections with smooth page transitions
- **Fully responsive** — Works across desktop, tablet, and mobile
- **Zero dependencies** — No npm, no build tools, just open and go
- **Custom aesthetic** — Warm cream & terracotta palette with Fraunces serif typography
- **Animated transitions** — Subtle fade-up animations on page switch

---

## 📄 Pages

| Page | Description |
|------|-------------|
| **About** | Introduction, personal interests, internship experience, and education timeline |
| **Skills** | Technical skill cards + all certifications |
| **Projects** | Showcase of Leo, Naruto, and Carezz with tech stack details |
| **Contact** | Email, phone, GitHub, and LinkedIn links |

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (custom properties, grid, flexbox, animations)
- Vanilla JavaScript
- Google Fonts — [Fraunces](https://fonts.google.com/specimen/Fraunces) + [DM Sans](https://fonts.google.com/specimen/DM+Sans)

---

## 🚀 Getting Started

No installation needed. Just clone and open.

```bash
git clone https://github.com/your-username/portfolio.git
cd portfolio
open index.html
```

Or simply download `index.html` and open it in any browser.

## 🎨 Design Highlights

- **Color palette** — Cream `#f5f0e8`, Ink `#1a1510`, Terracotta `#c4603a`, Sage `#7a9e7e`
- **Typography** — Fraunces (display/headings) + DM Sans (body)
- **Layout** — Asymmetric two-column grids with generous negative space
- **Interactions** — Hover lifts, color transitions, and staggered card reveals
